1. 사용에 대한 제한

   본 게시판은 누구나 사용가능한 제한된 프리웨어입니다.
   따라서 본 프로그램은 재배포가 가능한 프로그램입니다.
   단 소스 수정 배포시에는 shop-wiz의 동의가 있어야 합니다.
   본 게시판의 사용으로 인한 데이터 손실이나 기타 어떠한 피해에 
   대해서도 책임을 지지 않습니다.


2. 필요사항
   Linux Server
   PHP
   MYSQL
   file 데이타 베이스와 php를 사용하였기 때문에 PHP및 웹서버가 설치되어 있는 곳이면 어디든

지 사용 가능합니다.

3. 변화된 사항
* 위즈보드의 모든 기능들은 게시판 스킨에 따라 다소 차이가 있을 수 있습니다.
  1. 관리자 모드 기능강화
     - 등급별사용자 글쓰기, 리스트보기, 글읽기, 다운로드 제한기능(wizweb 이나 wizmall과 연동시 가능)
	 - 글작성시 포인트 기능(wizmall과 연동시)
     - 게시판 별 관리자 설정 가능
     - 파일업로드 허가 기능
     - 게시판 정열기능
     - 각페이지의 헤더 풋더 설정기능
     - 관리자 그룹기능
     - 삭제기능추가(만약을 고려해 wizweb이나 wizmall의 admin에서는 삭제기능이 없습니다.)
     - 스펨방지기능(스킨자체 및 금지단어설정기능)
     - 업로딩시 파일명 치환기능(한글이미지시 깨지는 것 방지용)
	 - 플래쉬와 연동

  2. 스킨기능
     - 스킨은 아이콘, 메인, 페이지표시, 리플스킨등 별도 분리
     - 또한 스킨별 관리자모드에서 색상변경이 가능하거나 크기변경이 가능합니다.(스킨의 특성

에 따라 다소 차이가 남)
  
4. 설 치
   
   - 아래의 디렉토리가 존재하지 않을시 직접만들어 주세요.--구버젼 알집의 경우 비어있는 디렉

토리를 없애는 경우가 있습니다.
   - 아래와 같이 텔넷에서 작업할 수 없는 경우 ftp에서도 퍼미션을 변경하실 수 있습니다.
     대부분의 경우 해당디렉토리에서 오른쪽 마우스를 클릭하면 권한변경에 관한 기능이 있습니

다.

   1). 일단 다운 받으신 파일을 푸시면 wizboard.php와 두개의 폴더(config, board)가 나타납니

다.
        ftp를 이용하여 모두 웹문서루트에 올려둡니다.

	public_html --- wizboard.php
        	     |
	             -- config
	             |
        	     -- wizboard --- table
                	       |
	                 	-- 다수의 폴더들

  2) 올리신 후 config(폴더), wizboard/table(폴더), wizboard/channel, util/wizmail의 퍼미션

을 각각 777 혹은 707로 변경합니다.

  3) 이 작업이 다되었어면
    "도메인/wizboard"를 웹브라우져상에서 타이핑하신후 지시사항에 따라 새로운 퍼미션 조정 및 

DB정보를 입력하면 자동으로 인스톨됩니다.

  4) 처음으로 넣는 패스워드와 아이디는 보드 Admin의 아이디와 패스워드이므로 분실하지 않도록 

유의하세요. 단, wizweb이나 wizmall을 설치시 각 관리자 모드에서 수정가능합니다.




5. 스킨관련
   현재 wizboard를 다운로드 받으시고 설치를 완료하면 제공되는 기본 스킨은 아래와 같습니다.

	public_html --- wizboarboard --- icon --- default
        	                      |
                             	       -- skin --- default
	                              |
        	                      -- skin_btnm --- style1
                	              |
	                              -- skin_reple --- default

	1) icon : 저장하기, 보기, 쓰기등등 각종 버튼과 관련한 아이콘들이 들어가 있습니다.

	2) skin : 가장중요한 것으로 이 보드의 스킨에따라서 제공되는 기능이라든지 게시판의 

용도가 결정됩니다. 현재 기본셑업과 함께 제공되는 default 스킨은 모든 기능을 포함하는 일반 

게시판입니다. shop-wiz에서 보시는 이 게시판도 default에서 약간 변형시킨 것입니다.

	3) skin_btnm : 리스트하단의 페이지표시 스킨입니다.

	4) skin_reple : 리플스킨으로 단순 Re기능이아니라 view mode에서 글 하단에 리플다는 

스킨입니다.

	가각의 스킨들은 업데이트되는 즉시 위즈보드게시판에 게재하고있으며 지금은 약 3종 정

도의 게시판스킨들이 있으며 이들은 각각 조합이 가능합니다.

	숍위즈의 위즈보드 스킨게시판에서 각각 필요한 스킨을 다운받으신 후 ftp를 이용해 각각

의 스킨 디렉토리 밑으로 넣으시면 자동으로 인식됩니다.

 

6. 관리자 모드 로그인
관리자 모드는 
여러분의 URL/wizboard 를 타이핑하시면 로그인창이 나타납니다. 이곳 로그인 창에서 인스톨시 넣

었던 아이디와 패스워드를 넣어시면 관리자 메인 창으로 들어갑니다. 일단 설명이 많이 필요한 부

분이라..
