3. 설치 방법

  ㅁ snapoll_config.php3 파일을 메모장으로 열고 아래부분을 수정합니다.

    $host = "localhost"; // mySQL 서버 주소

    $mysql_id = ""; // mySQL 아이디

    $mysql_pass = ""; // mySQL 패스워드

    $mysql_dbname = ""; // mySQL DB 이름

    $_SnaPoll_Admin = ""; // 관리자 패스워드 ( 대소문자 구별 )

    $path = "."; // 투표가 있는 곳의 주소

  ㅁ 이제 다 설정하셨으면 자신의 계정에 올립니다.

    http://...주소../snapoll_admin.html 로 접속합니다.

    관리자 패스워드를 입력 하시면 설치 화면이 나옵니다. 
  
    설치 버튼을 누르면 설치가 완료됩니다.



4. 사용 방법

  ㅁ 관리자 모드에서 투표를 생성합니다.

     http://..주소../snapoll.html?id=생성한 코드명

  ㅁ 홈페이지 속에 넣기

     넣고 싶은 자리에 아래 소스를 넣어주시면 됩니다.

     <?
     $id = "투표 코드";
     $bar_width = " "; // 투표 결과에 나올 막대 너비
     $bar_height = " "; // 투표 결과에 나올 막대기 높이
     $bar_color = " "; // 투표 결과에 나올 막대기 색
     $width = " "; // 투표 결과에 나올 표 크기
     $window_width = " "; // 투표 창 너비
     $window_height = " "; // 투표 창 높이
     include"snapoll.html";
     ?>
     
     $bar_width ~ $window_height 는 기본 값을 쓰시려면 아래처럼 지우셔도 됩니다.

     <?
     $id = "투표 코드";
     include"snapoll.html";
     ?>
